# 🎮 Arcaludo

Arcaludo est une application permettant aux utilisateurs de gérer, organiser et suivre leur collection de jeux vidéo.
Elle propose un système d’authentification sécurisé, une base de données structurée, une interface moderne en mode sombre et une intégration avec l’API RAWG.  
**TPI 2165** — Adrian Toledo, ETML 2026

---

## Stack

Node.js + Express · MySQL · JWT · API RAWG.io · Docker

---

## Fonctionnalités

- Inscription et connexion
- Rechercher des jeux sur RAWG
- Gérer ta collection (statut, note, heures, plateformes)
- Liste de souhaits
- Communauté 
- Supprimer ton compte
